# Code_Crusaders_75

Project Overview

An intelligent system designed to help users find nearby pharmacies with a required medicine in stock, compare prices, and receive smart recommendations based on availability, distance, and cost.
It also supports reading prescription images using AI-based OCR and displays pharmacies on an interactive map for easy navigation.
